一. 必须理解的两个概念

  1. code repository是代码仓库，我们把包代码上传到这个仓库。
  2. spec repository是配置仓库，所有的配置按照包名、版本号分门别类的存放在这个仓库。这个仓库只用来存放spec文件，不存放代码。
